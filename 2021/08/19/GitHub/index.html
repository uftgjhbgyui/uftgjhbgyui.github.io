关于GitHub

1.新建GitHub账号

  • 访问GitHub官网

https://github.com/

  • 进入官网在右上角找到这个
  • 这里写你的邮箱 QQ邮箱也可以
  • 邮箱验证成功后 会让你输入密码
  • 在这里填写你的用户名
  • 这里是否希望通过电子邮件接收产品公告，’y’表示是，’n’表示否
  • 之后会有一个验证进行真人验证，选出旋涡星系认准这张图
  • 验证成功后会给你的邮箱发送一份验证码 输入验证码我们的GitHub账号就创建成功了

2.创建GitHub仓库

  • 这两个地方都可以创建我们的GitHub仓库

3.创建GitHub项目

  • 给自己的仓库起一个名字，以后用来项目文件的上传

4.克隆仓库到本地

  • 我们可以选择https方式或者ssh方式来克隆自己仓库的地址
  • https方式会让你在上传的时候进行授权，如果不授权的话会报错，我们就会上传不上去。这时候就得配我们的ssh秘钥和公钥了

  • Win+R键输入cmd打开电脑命令框，输入ssh-keygen -t rsa -C ‘添加公钥的GitHub邮箱’（就是你注册时候用的邮箱）然后我们一直按回车键，直到命令执行结束

  • 这时候会生成一个文件夹叫（.ssh） 找到这个文件夹（我的电脑>c盘>用户>（这里找到你电脑里自己的文件夹）>.ssh）这个文件夹里有两个文件，我们选择第二个文件打开复制里面全部代码。

  • 在GitHub个人头像里找到设置

  • 在设置里找到SSH ard GpG Keys

  • 配置我们的公钥SSH

  • 这里我们起名ssh

  • 下边这里我们复制上刚才复制的.ssh文件里的代码，点击绿色按钮就完成了ssh公钥的配置

  • 这时候我们只要在克隆的时候选择ssh方式下载就可以了，会自动绕开秘钥验证

5.提交文件到仓库
